# Studio's Design System

Howdy! Here you can find our interface guidelines, component and theme documentation, and resources for building apps with Studio's design system.

### Background

Why is a design system necessary?

- Creates consistency and a unified visual language between cross functional teams
- Provides a foundation to build apps quickly and consistently
- Serves as a source of truth to reference design values, component documentation, and best practices
