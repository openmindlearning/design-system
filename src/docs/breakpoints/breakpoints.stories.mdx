# Breakpoints

Use the `ScreenSize` enum for standard breakpoints.

```ts
export enum ScreenSize {
  sm = 640,
  md = 768,
  lg = 1024,
  xl = 1280,
  xxl = 1536,
}
```

### In VanillaExtract styles:

```ts
import { style } from "@vanilla-extract/css";
export const myStyle = style({
  ...
  "@media": {
    ...atScreenSizeSm({
      justifyContent: "flex-start",
      padding: "0px 24px",
    }),
  },
});
```

### In React components:

```ts
const isScreenSmall = useBreakpoint(ScreenSize.sm);

return (
  {isScreenSmall && <></>}
)
```

_Note: `useBreakpoint` hook needs access to `window` property and will run client-side only, so `useBreakpoint` will default to `true` during SSR or SSG rendering._
